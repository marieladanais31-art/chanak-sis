# CHANAK SIS (Student Information System)

A comprehensive frontend React application for CHANAK International Academy, integrated with Supabase for authentication and data storage.

## Features

- **Role-Based Access Control**: Student, Parent, Tutor, Admin, Super Admin.
- **Hub Management**: Manage educational hubs and student assignments.
- **User Management**: Create and manage users directly from the dashboard.
- **Progress Reports**: View and manage student academic progress.

## Setup & Configuration

### Prerequisites
- Node.js v20+
- NPM

### Installation
1. Clone the repository.
2. Run `npm install`.
